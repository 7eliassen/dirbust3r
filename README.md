dirbust3r is a tool designed to scan a given URL for directories and files. It can be used to find hidden directories and files, and can also be used to check for common file extensions.

### Usage
To use dirbust3r, you can use the following command:

`python dirbust3r.py -u <url> [-w <wordlist>] [-r] [-v] [-t <threads>]`

You can use any wordlist you want.
